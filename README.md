# 五筆字型

配方： ℞ **wubi**

这是一个基于 [Rime](https://rime.im) 五笔字型输入方案，核心特点如下：

---

## 核心功能

- **86 五笔基础**：基于标准 86 五笔编码，部分码位优化，支持简码、词组输入。
- **简繁切换**：可一键切换简体与香港繁体（通过 `Control + Shift + 4`）。
- **拼音反查**：输入 `/拼音` 可反查五笔编码，辅助学习或临时输入。
- **英文输入**：内置英文词库，支持中英文混合输入与自动空格。
- **日期时间快捷输入**：支持 `rq`（日期）、`sj`（时间）、`xq`（星期）等快捷输入。
- **财务数字快捷输入**：支持 `Shift+英文字母`，实现大写会计数字。
- **Emoji 支持**：可输入常用 Emoji 表情，并支持降频调节。

---

## 特色功能

- **万能 Z 键**：支持用 `Z` 代替未知编码，自动匹配可能字符。
- **重复上屏**：输入 `z` 可重复上一次输入的内容。
- **英文单词降频**：自动将容易误输入的英文单词（如 `and`、`the` 等）降低优先级。
- **自动补全与上屏**：四码及以上时自动上屏，支持编码补全提示。
- **用户词典支持**：可记录个人常用词，动态调频。

---

## 操作与快捷键

- **候选选择**：可用 `;` 选第二项、`'` 选第三项。
- **翻页**：`Tab` 向下翻页，`Shift + Tab` 向上翻页。
- **回车换行**：在输入状态下按 `Enter` 直接换行。
- **中英标点切换**：`Control + ;` 或 `Control + Shift + 3`。
- **简繁切换**：`Control + Shift + 4`。

---

## 词库与配置

- **主词库**：`wubi`
- **辅助词库**：`cn_dicts`目录下
- **自定义短语**：`custom_phrase.txt`
- **拼音反查词库**：`pinyin_simp`
- **英文词库**：`English` 配置常用5000词，并大多数计算机、英文简写常用术语

---

## 手动加词及词频调整

- **词频调整**：手动调整`phrase_weight.txt`，再使用`replace_weight.py`进行词频替换，可永久调整词库词频
- **手动加词**：提供多种编码规则，启用`wubi.encoded.py`，按提示操作

---

## 适用场景

- 适合习惯使用 **86 五笔**、希望输入法**简洁高效**、同时需要**拼音辅助**、**英文混输**、**日期快捷输入**、**数字大写快捷输入**等进阶功能的用户。
- 如果需要进一步自定义，可编辑相应的 `custom_phrase.txt` 词库文件或调整 `wubi.schema.yaml` 中的参数。
